# site/

Static, dependency-free pt-BR pages for Curupira: the public privacy policy
(`privacidade.html`, required by the Play Store listing) and a minimal landing
stub (`index.html`).

These are published to the public repo `syst3mm/curupira-site` via GitHub Pages
**only at go-live** — see `docs/operations/beta-runbook.md` step 5. Do **not**
publish before that step.
